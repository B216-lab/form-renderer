# Решения

- Почему вся форма тупым гигантским компонентом?
  > Планируется хранить схемы форм в БД, имея возможность создавать любые формы, не лезя в исходники и не
  > проворачивая весь цикл обновления сервиса на проде ради небольшого изменения анкеты, для этого намеренно
  > здесь держал всю форму как можно больше в одном месте, не декомпозируя всё по-красивому, потому как в перспективе это никак не понадобится
